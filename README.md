# Power BI E-commerce Analytics Dashboard

## Project Overview
Interactive Power BI dashboard analyzing sales, profit, customers, and returns.

## Dataset
Sample Superstore Dataset

## KPIs
- Total Sales
- Total Profit
- Profit Margin
- Orders
- Returned Orders

## Dashboard Features
- Sales Trend Analysis
- Category Performance
- Regional Analysis
- Customer Segment Analysis
- Return Analysis

## Tools Used
- Power BI
- Excel/CSV
- DAX
- Power Query
